1. e-ROUTES App for EV Cars co-pilote

My Role

Junior UX Designer responsible for benchmarking, research, and design proposals for map-based visualizations.

The Problem

EV drivers needed a more intuitive way to locate and navigate to charging stations, with clear information about availability and compatibility.

The Solution

Developed map-based visualizations with comprehensive charging station information and optimized route planning for electric vehicles.

The Process

  • Conducted four comprehensive benchmarks of industry solutions
  • Identified critical use case patterns for EV charging
  • Created and presented actionable insights
  • Developed original design proposals
  • Refined UI/UX based on user feedback

2. Space App - A Unified Platform

My Role

UX Designer and Researcher leading usability testing for core features across multiple brands.

The Problem

Managing 14 separate brand-specific apps created fragmentation and inconsistent user experiences across the Stellantis portfolio.

The Solution

A unified platform that consolidates all brand apps while maintaining brand identity and improving core functionalities.

The Process

  • Conducted over 40 usability tests
  • Enhanced ADAS, EV Scheduling, and Remote Control features
  • Provided comprehensive feedback reports
  • Implemented improvements based on user testing

3. UX and UI Design for Stellantis Multibrand Design System

My Role

UI/UX Designer and Brand Skinner for the unified design system, focusing on Opel brand implementation.

The Problem

Need for a cohesive design system that maintains brand identity while ensuring consistency across multiple vehicle interfaces.

The Solution

Developed a flexible design system that adapts to brand-specific requirements while maintaining core functionality consistency.

The Process

  • Created wireframes and mockups for core functionalities
  • Collaborated with senior designers and engineering teams
  • Customized visual style, typefaces, and motion design for Opel
  • Ensured consistency with established component architecture

4. UX Design for Citroën Ami Mobile App

My Role

Lead UX Designer for the My Ami Play radio interface integration.

The Problem

Need to integrate complex radio functionality (DAB/FM) into a simple, user-friendly mobile interface.

The Solution

Created an intuitive interface that simplifies radio control while supporting both DAB and FM technologies.

The Process

  • Analyzed user requirements for radio functionality
  • Designed user-centered interface solutions
  • Integrated with vehicle systems
  • Successfully deployed in Citroën Ami vehicles

Each project has broadened my skills in UX design, research, and collaborative problem-solving across a variety of digital and automotive interfaces. I'm proud to have contributed to creating valuable, user-centered solutions that enhance Stellantis's digital ecosystem.
